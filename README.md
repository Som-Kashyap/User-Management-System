# User Management System

A console-based CRUD application that allows users to create account, login , edit details, fetch data , delete account. 
Has a menu-driven interface and different user-operations.  
It is integrated with MySQL database for persistence, allows users to save and load their data.

## Features

- Create Account / Signup
- Secured-Login ( checks for correctness of email and password )
- Edit account details ( Username / Password / id / Email / Gender )
- Fetch account details
- Delete account
- Menu driven, OOP and modular design

## Technologies

- C++
- MySQL
- Object Oriented Programming
- Git

## Future Improvements

- Add Admin mode and operations ( view / edit / search / delete users )
- Use prepared statements rather than hard-coded queries
- Split the project into modules for better readability and easier debuggin
